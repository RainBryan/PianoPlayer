Sheet Notation

By default, every character in a sheet is a literal key to press. Turn on Musician Notation in the Edit tab to use these symbols:

SymbolMeaningExample
c d e Single notes, played in sequence c d e f g
[ec] Chord — keys pressed together [ceg]
- Rest (silence) — one beat per dash c - d --- e
c= Hold — c rings under next notes c== d e f
c~ Fermata — freezes song while c is held c d e~~ f
c. Staccato — very short press c. d. e. f.
(160) Tempo change mid-song c d (240) e f
| Bar line (visual only) c d e f | g a b c
: :x3 Repeat section 3 times : c d e f :x3
